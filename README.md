## 博客简介
使用 Github Pages 和 Hexo 博客框架搭建的静态博客站,使用 redefine 主题,图床和网页代码使用 GitHub 仓库进行托管.

## 关于本站
本站用于记录在日常学习和瞎鼓捣时所遇到的问题,并提供一份个人的解决方案,希望能为你提供帮助.

本站目前还处在建设之中🏗,不过基本会涵盖以下几个大类的文章

* 网络安全
* HomeLab 搭建
* Linux 服务器运维
* Cisco 设备配置
* ESXI
